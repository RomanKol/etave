Settings

Here you can adjust the settings for the recordings.

Mouse events

The mouse events includes mousedown, mouseup, click, mousemove and mouseover events.

Key events

The key events includes keydown and keyup events.

Scroll event

The scroll events includes scroll events of any source (keyboard, scrollwheel, scrollbar) of the window.

Change event

The change events triggers when the users interacts with input elements like radio buttons, selections, checkboxes oder text inputs.

DOM events

The dom events includes changes in the dom like adding/removing elements, changing attributes (classes, data) or changing nodes.

Throttling

To prevent too much data beeing recorded, mousemove and scroll events are throttled by time and distance.
